# 春江花月夜

<!-- more -->

::: info 简介
作者: 张若虚
:::


::: center

春江潮水连海凭，海上明月共潮生。

滟滟随波千万里，何处春江无月明。

江流宛转绕芳甸，月照花林皆似霰。

空里流霜不觉飞，汀上白沙看不见。

江天一色无纤尘，皎皎空中孤月轮。

江畔何人初见月？江月何年初照人？

人生代代无穷已，江月年年望相似。

不只江月待何人，但见长江送流水。

白云一片去悠悠，青枫浦上不胜愁。




:::
