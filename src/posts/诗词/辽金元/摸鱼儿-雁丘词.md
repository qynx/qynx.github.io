# 摸鱼儿-雁丘词

<!-- more -->

::: tip
[元好问](../../诗人/元好问.md)
:::


::: center 

问世间，情是何物，直教生死相许？

天南地北双飞客，老翅几回寒暑。

欢乐趣，离别苦，就中更有痴儿女。

君应有语：渺万里层云，千山暮雪，只影向谁去？

横汾路，寂寞当年箫鼓，荒烟依旧平楚。

招魂楚些（suò）何嗟及，山鬼自啼风雨。

天也妒，未信与，莺儿燕子俱黄土。

千秋万古，为留待骚人，狂歌痛饮，来访雁丘处。

:::
